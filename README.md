# Pokedex

_Proyecto que consume la API de Pokemon, los lista y muestra sus detalles._

### 🔧 Instalación

_En el directorio del proyecto, puede ejecutar:_

```
npm i
```
```
npm run dev
```

## 📦 Resultados

_También puedes ver el proyecto en:_

[Pokedex](https://kalevins.github.io/pokedex/)
